Add a splash page to all entry pages to encourage donations to the Red Cross for disaster relief

== Description ==

Adds a splash page to all entry pages (any page on your site) to encourage donations to the Red Cross for disaster relief. You can control the messaging, and the splash page is permanently dismissed after one viewing.

**Credit to** the work of Yaron Schoen and Noah Stokes. [More information](https://github.com/noahstokes/Sandy-Relief/tree/Moore-Relief).

== Installation ==

1. Download the plugin and extract the files
1. Upload `relief-splash-page` to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
